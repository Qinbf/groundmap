# De Minimis 免税 ^h-1-1-f31b17

> 美国 De Minimis 规则曾允许**单日单人 < $800** 进口包裹**免关税 + 免正式海关申报**。**2025-08-29 全面终结**[[raw/articles/taxcloud_de_minimis_end_2025#^p-66-e8a801]]——这是过去十年跨境电商最大的政策变动。 ^p-1-5ce3f4

## 政策时间线 ^h-2-1-a9fa38

[[wiki/sources/shipbob_de_minimis_adapt#^p-39-85500c]]： ^p-2-c630c7

| 时间 | 事件 |
|---|---|
| 2016 | TFTEA 法案把 De Minimis 从 $200 提到 $800 |
| 2023-24 | 国会与贸易团体施压改革（理由：非法贸易 + 关税规避 + 中国占比过高） |
| 2024-04 | Biden 政府提议加强数据要求 + 执法 |
| **2025-05-02** | 中国 + 香港 Section 321 资格取消 |
| **2025-08-29** | **全球**所有低值包裹必须正式申报 + 关税 |
| 2026-02-28 | 邮政过渡期结束，全部走 ad valorem 从价税 |
^t-1-dme01 ^p-3-05d342

## 新规则（2025-08-29 起）^h-2-2-dme01 ^h-2-2-e6d490

- 所有进口包裹（无论金额）**全部缴税 + 走 ACE 正式申报**
- 承运 / 报关行须递交 10 位 [[wiki/concepts/hts_code]] HTS 编码
- 过渡期内（至 2026-02-28）邮政低值包裹可选 **$80-$200/件定额**关税；之后全部 ad valorem [[wiki/sources/taxcloud_de_minimis_end_2025#^p-68-37b89c]]
- 多国邮政运营商已**暂停 / 减缓**对美寄递 [[wiki/sources/taxcloud_de_minimis_end_2025#^p-69-daa79d]] ^p-4-4f0683

## 谁被影响 ^h-2-3-52a479

按 TaxCloud 整理（[[wiki/sources/taxcloud_de_minimis_end_2025#^p-70-412e98]]）： ^p-5-d8c602

- **直邮 / dropshipper**：依赖低成本进口的卖家（[[wiki/entities/temu]] / [[wiki/entities/shein]] / AliExpress）
- **SMB**：跨境采购库存 / 样品的小企业
- **海外仓直发的全球品牌**
- "你以为自己不是进口商但货物跨境入美——你就被影响" [[wiki/sources/taxcloud_de_minimis_end_2025#^p-71-80f1b5]] ^p-6-313aa3

## 卖家应对（7 步行动）^h-2-4-dme01 ^h-2-4-b2f8fb

来自 [[wiki/sources/paypal_de_minimis_guide#^h-2-5-083a3e]]： ^p-7-2dc16b

1. 审计现有跨境业务流
2. 重新计算 [[wiki/concepts/landed_cost]]
3. 与承运 / 报关方对齐
4. 把关税并入标价 OR 给买家清晰提示
5. 评估是否在美国本土设仓（[[wiki/concepts/fba]] / 3PL）
6. 升级 ERP / 订单系统支持 HTS code
7. 沟通好客户运输预期 ^p-8-b10dec

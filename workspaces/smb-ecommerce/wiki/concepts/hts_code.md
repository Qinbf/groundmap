# HTS 编码 ^h-1-1-3d2ef1

> 美国海关进口商品的**10 位关税分类编码**——前 6 位与全球 HS 通用，后 4 位为美国细分。 ^p-1-ea0e6e

## De Minimis 终结后的强制性 ^h-2-1-77568e

[[wiki/sources/taxcloud_de_minimis_end_2025#^p-68-37b89c]]： ^p-2-6eb1b8

> "Carriers and brokers must file full customs entries with 10-digit tariff codes." ^p-3-115ad0

每件包裹（无论金额）都要在 ACE 系统申报正确的 HTS 编码，由此确定关税率。**归类错误**可能被处罚或货物滞留。 ^p-4-441318

## 卖家操作 ^h-2-2-99433d

- ERP / 订单系统需支持自动归类
- 必要时聘请专业归类员（customs broker） ^p-5-7422ed

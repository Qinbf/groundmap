# Temu ^h-1-1-9f671c

> **PDD Holdings**（拼多多母公司）的国际化产品 ^p-1-289021

## 关键事实 ^h-2-1-507e6a

- **2022-09 上线**美国市场，主打超低价 + 海量 SKU
- 上线后高速增长，在 2023-25 期间多次成为 App Store 下载榜首
- PDD Holdings 营收 2023 → 2024 → 2025：RMB 247.6B → 393.8B → 431.8B（Temu 为重要驱动）
- 受 [[wiki/concepts/de_minimis_exemption]] 终结**直接冲击**——三大 De Minimis 来源（[[wiki/sources/taxcloud_de_minimis_end_2025]] / [[wiki/sources/shipbob_de_minimis_adapt]] / [[wiki/sources/paypal_de_minimis_guide]]）都把 Temu 列为首要受影响平台 ^p-2-54c09a

## 模式 ^h-2-2-f0789e

- 中国小批量直邮 + 跨境物流补贴
- 重度依赖美国 [[wiki/concepts/de_minimis_exemption]]（$800 阈值）
- 半托管模式：商家发货到 Temu 美国仓后由 Temu 完成最后一公里 ^p-3-406c3c

# Shopify ^h-1-1-329a01

> [[wiki/concepts/independent_site]] 模式的**主流基础设施**，是中国卖家做 D2C 出海的首选。 ^p-1-fc1c14

## 关键事实 ^h-2-1-507e6a

- SaaS 模式：建站 + 收单 + 物流 + 流量
- 与 Amazon 等第三方平台**互补**：Shopify 卖家保留客户数据与品牌掌控权
- **被点名为 De Minimis 终结后的工具方**：[[wiki/sources/taxcloud_de_minimis_end_2025]] 把 Shopify 列为主要集成对象之一
- 服务于 [[wiki/sources/modern_retail_tiktok_smb_66]] 中提到的 Whatnot 直播平台等基础设施 ^p-2-7d64c3

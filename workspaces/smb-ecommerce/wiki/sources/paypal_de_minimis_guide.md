# PayPal：De Minimis 终结后的卖家指南 ^h-1-1-609dfd

> **来源**：PayPal Business Resource Center（BRC）· **2025**
> **原始页面**：[[raw/articles/paypal_de_minimis_guide]] ^p-1-3fca5b

## 核心论点 ^h-2-1-522a48

1. **政策事实**（[[raw/articles/paypal_de_minimis_guide#^h-2-1-aefc81]]）：与 [[wiki/sources/taxcloud_de_minimis_end_2025]]、[[wiki/sources/shipbob_de_minimis_adapt]] **一致** —— 2025-08-29 全部国家失去 $800 免税。
2. **豁免范围**（[[raw/articles/paypal_de_minimis_guide#^h-2-2-ab56c4]]）：私人礼物 / 个人物品 / 部分类别仍可豁免；商业进口全部纳入。
3. **对卖家影响**（[[raw/articles/paypal_de_minimis_guide#^h-2-3-661c60]]）：定价、库存策略、客户体验三方面均需调整。
4. **关税并入定价**（[[raw/articles/paypal_de_minimis_guide#^h-2-4-ef735d]]）：把关税成本前置到商品标价中，避免买家收到关税账单产生 chargeback。
5. **7 步行动清单**（[[raw/articles/paypal_de_minimis_guide#^h-2-5-083a3e]]）：
   - 审计现有跨境业务流
   - 重新计算落地成本
   - 与承运 / 报关方对齐
   - 把关税并入标价 OR 给买家清晰提示
   - 评估是否在美国本土设仓
   - 升级 ERP / 订单系统支持 HTS code
   - 沟通好客户预期（运输时长、可能延迟） ^p-2-e799d7

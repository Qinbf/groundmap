# $800 De Minimis 之后：后续 Fulfillment 策略 ^h-1-1-a6869f

> **来源**：ShipBob Blog · **2025-09 发布**
> **本文规模**：32,190 字符（**第②档**），按 H2 切块阅读 ^p-1-e0b689

## 核心论点 ^h-2-1-522a48

1. De Minimis 阈值（< $800 美元免税）是过去十年跨境电商爆发的关键**杠杆**——TFTEA 2016 把美国阈值从 $200 提到 $800，使其成为全球最宽松 [[raw/articles/shipbob_de_minimis_adapt#^p-39-85500c]]。
2. **时间线**（key dates） [[raw/articles/shipbob_de_minimis_adapt#^p-39-85500c]]：
   - 2016：TFTEA 把 De Minimis 从 $200 提到 $800
   - 2023-24：立法与贸易团体施压，关注"非法贸易、关税规避、中国占比过高"
   - 2024-04：Biden 政府提议加强数据要求与执法
   - **2025-05-02**：中国 + 香港 Section 321 资格被取消
   - **2025-08-29**：全球 De Minimis 全部废除
3. **对 SMB 卖家的三重冲击** [[raw/articles/shipbob_de_minimis_adapt#^p-31-a5551f]]：
   - 落地成本（landed cost）抬升
   - 合规要求增加（formal entry / HTS code）
   - 供应链规划复杂度上升
4. **关键 takeaway** [[raw/articles/shipbob_de_minimis_adapt#^p-22-505eb3]]：De Minimis 是国家级阈值，跨境电商一直依赖它做"成本节约 + 市场扩张 + 供应链效率"——一旦消失，整个运营模型要重构。 ^p-2-e3ff71

## 应对策略（节选） ^h-2-2-1d8fe0

来自正文 H2「What de minimis values mean for ecommerce businesses」与「Strategies for adapting」节（[[raw/articles/shipbob_de_minimis_adapt#^h-2-5-9c514c]]）： ^p-3-915b53

- **加大美国本土仓储**：把海外仓策略转向**美国境内 FBA / 3PL**，缩短最后一公里
- **重算落地成本**：包含关税 + 报关费 + 时间成本，重新定价
- **HS/HTS 分类专业化**：准确归类避免被高税率打击
- **小包改商业批次**：从 small parcel 切换到 LCL / FCL，摊薄关税 ^p-4-8867c7

# Support

For general usage questions, use GitHub Discussions once the public repository is live.

For bugs, open an issue with:

- your operating system,
- Python version,
- Node.js version,
- the exact command you ran,
- the relevant error output,
- whether you are using the Web console or CLI.

For security issues, see [SECURITY.md](SECURITY.md).

For private deployments, industry playbooks, and product integrations, keep those discussions separate from public issue threads if they contain business-sensitive information.

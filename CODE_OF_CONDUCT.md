# Code of Conduct

GroundMap aims to be a thoughtful, technical, and welcoming open-source project.

## Expected Behavior

- Be respectful and precise.
- Assume good intent, but do not ignore harm.
- Critique ideas, designs, and code rather than people.
- Give context when disagreeing.
- Keep discussions useful for future readers.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated low-effort disruption after maintainers ask for a change.
- Using project spaces to promote spam, scams, or unrelated products.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or take other reasonable actions to keep the project healthy.

If you need to report a problem, use the private security/advisory channel once the public repository is created, or contact the maintainer through the project's published contact method.

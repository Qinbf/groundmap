11ANSWER把冲突诚实摆出来

Agent 最终回复

TikTok 母公司不公开美国 GMV，行业靠两个第三方估算：

  • Emarketer 2025-12-09 预测：$15.82B（+108% YoY，占社交电商 18.2%） [[raw/articles/emarketer_tiktok_social_commerce#^p-136-394777]]
  • Charm.io 估算（Modern Retail 引用）：~$14B [[raw/articles/modern_retail_tiktok_smb_66#^p-40-2ff788]]

差异约 13%。两个口径都不精确，但对 SMB 卖家决策无显著影响——不论 $14B 还是 $16B，平台都是 SMB 必争之地：215K 活跃 SMB（+25% YoY）、SMB GMV YoY +66%。

方法学差异详见 [[wiki/analyses/tiktok_shop_us_market_size_2025]]。

⚠️ 传统 chunk-RAG 会怎样

1. embedding 召回会把"$15.82B"和"$14B"两个 chunk 都拿到 — 但**没有结构告诉 LLM 这是冲突，不是补充信息**

2. LLM 综合时倾向"看起来更权威"的来源（Emarketer，因为有"forecast"措辞）— 把另一个数字 silenced

3. 用户看到的是"$15.82B"这个单一数字，不知道行业其实分歧

4. 一旦用户用这个数字做投资估值或合同，风险落到用户头上

Agent 最终回复

如果你看重「冷启动 + 流量获取」

→ TikTok Shop 优先。美妆是平台第一类目（占 19% GMV / $2.7B 2025），67% 消费者用 TikTok Shop 做品牌发现，超过 Amazon。SMB 平均年增 66%。[[wiki/entities/tiktok_shop]]

如果你看重「政策稳定性 + 物流成熟」

→ Amazon FBA 更稳。De Minimis 终结后，Amazon FBA 受冲击远小于 TikTok Shop / SHEIN / Temu 的直邮模式，因为是商业批次入境。[[wiki/concepts/de_minimis_exemption]]

务实做法（综合判断）

  • 两个都做、各占 50%：TikTok Shop 用于冷启动 + 内容种草；Amazon FBA 用于沉淀 + 复购
  • 本土仓策略：先发一批货到 FBA / 3PL 美国仓，TikTok Shop 订单走美国本土发货——同时绕过 De Minimis 关税 + 缩短最后一公里
  • 跟踪政策窗口：邮政 $80-$200 定额过渡到 2026-02-28，之后全部 ad valorem，重算落地成本 [[raw/articles/taxcloud_de_minimis_end_2025#^p-68-37b89c]]

4 core design principles

No vector recall
Retrieval is BM25 full-text + metadata filtering. No vector store, no chunking.
Block-level anchors
Every paragraph carries a stable anchor (e.g. ^p-12-7d8e9a), so citations land on the exact paragraph.
Conflict is visible
Diverging sources are not averaged; they are kept explicit with [!WARNING] blocks.
Three-tier ingest
Short docs are read in one go, medium ones split by H1, long ones get a TOC scan plus a per-chapter depth log.
